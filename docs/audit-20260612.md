**D-3. Idempotency 판정**

| 구문 | 방식 | Idempotent? |
|---|---|---|
| `CREATE TABLE IF NOT EXISTS` | IF NOT EXISTS ✓ | ✅ |
| `CREATE INDEX IF NOT EXISTS` | IF NOT EXISTS ✓ | ✅ |
| `ALTER TABLE ... ENABLE ROW LEVEL SECURITY` | 중복 실행해도 에러 없음 | ✅ |
| `CREATE POLICY "..."` | `IF NOT EXISTS` 없음 | **❌ 재실행 시 오류** |

→ `CREATE POLICY` 3개 파일 총 5개가 `IF NOT EXISTS` 없음. 동일 migration을 두 번 적용하면 `policy "..." already exists` 오류 발생.

```typescript
      reply_markup: buildStartKeyboardMarkup(webAppUrl),
    }),
  });
}
```

- `webAppUrl = ${appBaseUrl}/webapp` — **`process.env.APP_BASE_URL` 기반** ✅
- HTTPS 검증: `sendStartKeyboard` 함수 자체에는 없음. `route.ts` line 67에서 호출 전 `appBaseUrl.startsWith("https://")` 검증 수행 ✅

**F-3. `.env.local` APP_BASE_URL 현재 값**

E항 실측 결과와 동일 — `.env.local`에 `APP_BASE_URL` 키 **없음(미설정)**.

`.env.example`의 기본값은 `APP_BASE_URL=http://localhost:3000`이나, `.env.local`에 실제 HTTPS URL은 설정되어 있지 않다.
