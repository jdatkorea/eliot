## 2. 사용자 경험(UX) 및 유저 여정(User Journey)

### 2.1 전체 플로우 (코드 추적)

```mermaid

    participant TG as Telegram Bot

    participant REL as deliverTripBriefing
    participant SB as Supabase briefings

    participant FB as /feedback
    participant FE as POST /api/feedback

    U->>TG: /start
    TG->>WH: Telegram Update
    WH->>TG: sendStartKeyboard (web_app → /webapp)

    REL->>SB: saveBriefingPayload (A+B payload)
```

## 부록: 현재 코드에서 확인된 주요 제약·특이사항

5. **레이아웃 metadata·admin ID**는 프로덕션 미정비 상태로 보이는 placeholder 값이 코드에 남아 있음 (`layout.tsx` L15–17, `BriefingView.tsx` L51).
